# وميض — دليل التثبيت على ويندوز

## كيفية الاستخدام بعد التثبيت

1. شغّل **Wameed** من قائمة ابدأ أو من سطح المكتب.
2. ستظهر نافذة حالة الخادم:
   - 🔵 **الخادم يعمل • بانتظار الهاتف** ← جاهز لاستقبال الملفات.
   - 🟢 **الهاتف متصل الآن** ← يوجد هاتف مرتبط فعلياً.
   - 🔴 **خطأ** ← المنفذ مستخدم أو مشكلة في الشبكة.
3. على هاتفك الأندرويد، افتح تطبيق **وميض** — سيكتشف الكمبيوتر تلقائياً.
4. اختر أي ملف/صورة/رابط وشاركه عبر **⚡ وميض** → يفتح على الكمبيوتر فوراً.

## ملاحظات مهمة

- **لا تحتاج Python**: البرنامج نسخة exe مستقلة.
- **Windows Firewall**: إذا اخترت الخيار أثناء التثبيت، تم فتح المنفذين 7788 (TCP) و7789 (UDP) تلقائياً.
- **التشغيل التلقائي**: الخيار اختياري في التثبيت — يمكن إلغاؤه من `shell:startup`.
- **SmartScreen**: عند أول تشغيل للمثبّت قد تظهر شاشة "Windows protected your PC" — اضغط **More info → Run anyway** (نعمل على توقيع الملف لاحقاً).

## إزالة التثبيت

من **Settings → Apps → Wameed → Uninstall**.
- تُحذف كل الملفات.
- تُزال قواعد Firewall تلقائياً.
- يُحذف اختصار Startup إن وُجد.

## استكشاف الأخطاء

| المشكلة | الحل |
|---------|------|
| الهاتف لا يرى الكمبيوتر | تأكد أنهما على نفس شبكة Wi-Fi + Firewall مسموح |
| `المنفذ 7788 مستخدم` | نسخة أخرى من وميض تعمل (تحقق من الشريط أسفل الشاشة) |
| الإرسال يتوقف في المنتصف | حدّث للنسخة الأحدث — هذه المشكلة أُصلحت |
